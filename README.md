# Chocolatey-Personal
This is my script to install my personal programs that I use most of the time

You are free to frok it!

## List of programs ##
- Steam
- Edge
- Vlc
- 7zip
- Epic Games Launcher
- Gimp
- Kdenlive
- HandBreak
- OBS
- Spotube
- Localsend
- qbittorrent
- vscode
- Foxitreader
  
## Script ##
```
choco install steam microsoft-edge vlc 7zip epicgameslauncher gimp kdenlive handbreak obs-studio spotube localsend qbittorrent vscode foxitreader -y
```

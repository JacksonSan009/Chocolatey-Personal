# Chocolatey-Personal
This is my script to install my personal programs that I use most of the time

You are free to frok it!

## List of programs ##
- Steam
- Edge
- Vlc
- 7zip
- Gimp
- Kdenlive
- HandBreak
- OBS
- Localsend
- qbittorrent
- vscode
- Foxitreader
- Free Download Manager
- Heroic games launcher
  
## Script ##
```
choco install steam microsoft-edge vlc 7zip gimp kdenlive handbreak obs-studio localsend qbittorrent vscode foxitreader freedownloadmanager heroic-games-launcher -y
```
